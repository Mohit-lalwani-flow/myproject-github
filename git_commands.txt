git init - initializes a new repository

git add . - adds all files to staging

git commit -m "message" - saves snapshot of changes

git branch feature - creates new branch

git checkout feature - switches branch

git merge feature - merges branch into main

git log --oneline - shows commit history

git reset --soft HEAD~1 - undo last commit

git remote add origin URL - connects to GitHub

git push -u origin main - pushes code to GitHub
